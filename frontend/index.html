How this system works
1
Download
Fetch scientific papers from Europe PMC on your chosen topic. PDFs and metadata are stored locally.
→
2
Process
Extract text from PDFs, split into overlapping chunks (~512 tokens). Required before any LLM task.
→
3
Run Tasks
Claude summarises, extracts structured data, or answers questions. OpenAI independently checks every output.
→
4
Evaluate
8 automated test categories (hallucination, grounding, numeric accuracy, …) run on a stratified 10 % subset.
→
5
Inspect Metrics
Pass rates, weighted risk score, and per-category breakdown reveal where the LLM fails.
